# GERÊNCIA

> Organiza o role, não passe vergonha.

Web app para squads de amigos organizarem encontros, double dates, saídas em grupo e festas exclusivas. Chega de bagunça no grupo do WhatsApp — aqui o date sai do papel.

## Features

- 🎯 **4 tipos de date**: Double Date, Grupo, Hunt Mode, Na Casa
- 📍 **35+ locais de Brasília**: Restaurantes High End, Mais Tranquilos e Bares
- 🔒 **Festas Exclusivas da Bolha** (premium)
- 👥 **Sistema de Squad** com código de convite
- 💰 **Controle de budget** automático
- 📱 **Mobile-first**, funciona no celular e desktop

## Stack

- HTML + CSS + JavaScript puro
- LocalStorage para persistência
- Zero dependências externas

## Deploy

Hospedado na Vercel.
