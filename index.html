Conectar Google Maps

Para buscar estabelecimentos fora da curadoria, adicione sua chave da API do Google Places. A chave é salva apenas no seu dispositivo.

  1. Acesse o Google Cloud Console
  2. Crie um projeto ou selecione um existente
  3. Ative a Places API e Maps JavaScript API
  4. Gere uma chave em Credenciais
  5. Restrinja a chave por HTTP referrer adicionando: gerencia-ten.vercel.app/*
  6. Cole abaixo
